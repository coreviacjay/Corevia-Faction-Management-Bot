# Corevia Faction Management Bot
A modular, production‑ready Discord bot designed for managing FiveM factions, inventories, finances, and member structures — all from Discord.  
Built with precision, scalability, and Corevia‑grade branding.


## 🚀 Features

### 🏛️ Faction Management
- Create and delete factions  
- Add and remove members  
- Promote and demote ranks  
- View faction summaries  
- JSON‑based persistent storage  

### 💰 Finance System
- Deposit and withdraw faction money  
- Track faction bank balances  

### 🔫 Inventory System
- Add/remove weapons  
- Add/remove drugs  
- Add/remove custom items  
- Full inventory breakdown via `/faction inventory`  

### 📊 Faction Overview
- `/faction about` shows:
  - Member count  
  - Total weapons  
  - Total drugs  
  - Total items  
  - Bank balance
